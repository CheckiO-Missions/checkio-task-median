So, how my final solution should look like?

You just need to wrap the code into function expression and return median.

Here is the whole solution:

def checkio(data: list[int]) -> int | float:

    items = sorted(data)
    ind, rem = divmod(len(items), 2)
    median = ((items[ind - 1] + items[ind]) / 2, items[ind])[rem]

    return median

If your half is 2 for example, you actually take the third element from the beginning of the list. ~ gives you the actual third element from the end of the list. For the list with the odd number of elements it will be the same element. Read more in this explanation. 

def checkio(data: list[int]) -> int | float:

    data.sort()
    half = len(data) // 2

    return (data[half] + data[~half]) / 2

There is statistics module in Python standard library with median function. It this solution, when you call checkio(data) the median(data) is called. 

from statistics import median as checkio
